# Task Manager App

## Descripción
Task Manager App es una aplicación web desarrollada con React y Vite que permite a los usuarios gestionar sus tareas de manera eficiente. Con esta herramienta, los usuarios pueden agregar, editar, eliminar y marcar tareas como completadas.

## Características
- Crear, editar y eliminar tareas
- Marcar tareas como completadas
- Filtrar tareas por estado (pendiente, completado)
- Interfaz intuitiva y responsiva

## Tecnologías utilizadas
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [React Hooks](https://react.dev/reference/react)
- [Tailwind CSS](https://tailwindcss.com/) (Opcional, si se usa para estilos)

## Instalación y uso
### Requisitos previos
Asegúrate de tener instalado [Node.js](https://nodejs.org/) y [npm](https://www.npmjs.com/) o [yarn](https://yarnpkg.com/).

### Pasos para instalar y ejecutar el proyecto
1. Clonar el repositorio:
   ```sh
   git clone https://github.com/usuario/task-manager.git
   cd task-manager
   ```
2. Instalar dependencias:
   ```sh
   npm install
   ```
   o con Yarn:
   ```sh
   yarn install
   ```
3. Ejecutar la aplicación en modo desarrollo:
   ```sh
   npm run dev
   ```
   o con Yarn:
   ```sh
   yarn dev
   ```
4. Abrir en el navegador: `http://localhost:5173/`

## Scripts disponibles
- `npm run dev`: Inicia el entorno de desarrollo con Vite
- `npm run build`: Genera la versión de producción
- `npm run preview`: Previsualiza la versión de producción
- `npm run lint`: Ejecuta ESLint para verificar el código
